Accessibility

Accessibility built in, not bolted on.

Which is why we'd retire the accessibility overlay on day one and rebuild accessibility from the theme layer out.
The same approach that took NASA.gov to 100% accessibility scores.

Layer 01
Theme-layer compliance.
WCAG 2.1 AA embedded in the theme itself — semantic markup, keyboard paths, ARIA roles, color contrast, focus states. Shipped with the build, not sprinkled on top.
Layer 02
Editor-level guardrails.
Equalize Digital Accessibility Checker Pro flags issues inside the block editor before content ships. Works across English and Spanish content.
